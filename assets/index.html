      About

      Autonomous Mesh CRDT Todo. Data is replicated peer-to-peer over your LAN using Ed25519-authenticated TCP and UDP beacon discovery. Deletions are tombstoned and propagate across nodes — to delete from all nodes, sign in on each one and delete there.

      Delete all tasks on this node?

      This marks every task as deleted on this node only. Peers will receive the tombstones on next sync. To remove data from all nodes, you must log in on each node and delete there individually.
